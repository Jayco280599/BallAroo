Control

The use of a turn dials and buttons would be designed to be as easy as possible that would require minimal effort, experience, and knowledge to use to its full ability. The turn dials can adjust the launch pitch and velocity as well as switches to turn on or off the unit and as well as the training function. A RGB LEDs would be used to alert the user what mode the unit is currently in. The system would have a self-training mode that can be selected with a switch, that will drop a treat into a bowl for the dog to enjoy whenever the ball is put into the feeder. With the self-training mode enabled, there would be a dwell time between the ball placed in the feeder and the ball being re-launched while the treats being dispensed. A commercially available Arduino or similar SBC (Small Board Computer) would be used as the smart controller of the system and provide the desired functionality of the unit. This would make prototyping the controller side of the product cheaper and simpler, as they are widely available with mature and supported toolchain. Initial prototype would be built using off-the-shelf general hardware components where possible, thus reducing initial cost and complexity. The alteration of operational parameters and performing software updates can be done directly using a cable USB and a host computer.
