Tools and Technology Overview

Overview

The prototype for the BallAroo would be constructed using as many off-the-shelf components as possible in effort to reduce construction time and minimise costs. This will facilitate testing of required technologies and desired feature set, while the final design is nailed down.

Software

As the BallAroo would be using the Arduino SBC as base, all the tools required to develop software for the BallAroo are opensource licensed using the GNU Lesser General Public License. This makes the software tools that are to be used for the project free. Applications like CAD (Computer Aided Design) will need to be used during the design phase, to enable production the individual components by 3D printing during the prototype phase, as well as sending the final designs for manufacturing. Autodeskâ€™s AutoCAD and Fusion360 are well known commercial products that are widely used across many industries. The Product Design and Manufacturing Collection from Autodesk can be used on a subscription basis with costs around $4,015 AUD per year for small teams. There are open-sourced alternates available such Libre CAD for 2D drafting and FreeCAD for 3D which are available at no cost. No matter which products are used, there may be a steep learning curve for those who are not familiar with computer aided drafting.
